# Plutonium Kidz Content Archive

This directory contains all the converted and organized content from the Plutonium Kidz archives. All files have been given self-documenting names for easy identification.

## Directory Structure

### 01-band-story/ (1 file)
Band origin story and history
- `band-origin-story.md` - The band's origin story involving radiation experiments creating Wonderwhirl, Molly Fantasatique, Rory Rocket, and Inspector Wong

### 02-songs/ (2 files)
Song information, stories, and technical notes
- `songs-descriptions-and-stories.md` - Introductions and descriptions for songs (Narcochamber, Chlorophyll Deficiency, Get Down, Surfin' Rabbi, etc.)
- `songs-technical-notes.md` - **MERGED FILE** containing all technical details: chord progressions, musical notation, guitar settings (V-amp), album track listing, and timing information

### 03-comedy/ (3 files)
Stage banter, comedy bits, and interesting facts
- `comedy-amazing-facts.md` - Collection of "True Facts" and "Useless Facts"
- `comedy-deep-thoughts.md` - Jack Handey-style satirical observations and absurdist humor
- `comedy-stage-banter.md` - Stage banter, jokes, "I pledge allegiance to the kidz", and commentary for live shows

### 04-setlists/ (7 setlists)
Historical setlists from various venues and performances
- `setlist-afternoon-delight-radio.md` - Radio show setlist with song stories and commentary
- `setlist-club-axis.md` - Club Axis performance
- `setlist-general.md` - General setlist
- `setlist-hi-pointe.md` - Hi-Pointe venue setlist
- `setlist-molly-full.md` - Molly's personalized full setlist
- `setlist-short.md` - **MERGED FILE** containing both Mike's and Molly's versions of the shortened setlist
- `setlist-wayout.md` - Wayout venue setlist

### 05-comic-book/ (2 files)
Comic book scripts and related content
- `comic-help-letter.md` - Letter about comic book creation
- `comic-script-revised.md` - Comic strip script (revised version)

## Source Files

Original Word documents (.doc files) remain in their original locations:
- `Documents/` - Original band documentation
- `Comic Book/` - Original comic book files and assets

## File Naming Convention

All files now use self-documenting names with the following patterns:
- **Band info:** `band-*`
- **Songs:** `songs-*`
- **Comedy:** `comedy-*`
- **Setlists:** `setlist-*`
- **Comic book:** `comic-*`

This makes it easy to identify file types and content at a glance.

## Usage

This organized content is ready to be integrated into the Plutonium Kidz website. See the main website integration plan for how to incorporate these files into HTML pages.
