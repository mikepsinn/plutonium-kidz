# Plutonium Kidz Content Archive

This directory contains all the converted and organized content from the Plutonium Kidz archives.

## Directory Structure

### 01-band-story/
Band origin story and history
- `Story of the Pu 239 Kidz.md` - The band's origin story involving radiation experiments

### 02-songs/
Song information, stories, and technical notes
- `song stories.md` - Introductions and descriptions for songs (Narcochamber, Chlorophyll Deficiency, Get Down, Surfin' Rabbi, etc.)
- `all mike's notes.md` - Technical details, timing, and guitar settings for songs
- `Guitar Settings.md` - V-amp settings for specific songs
- `CD track list (R.M. and others).md` - Track listing information

### 03-comedy/
Stage banter, comedy bits, and interesting facts
- `amazingfacts.md` - Collection of "True Facts" and "Useless Facts"
- `deepthoughts.md` - Jack Handey-style satirical observations and absurdist humor
- `stuff to say.md` - Stage banter, jokes, and commentary for live shows
- `I pledge allegience.md` - "I pledge allegiance to the kidz" and other bits

### 04-setlists/
Historical setlists from various venues and performances
- `afternoon delight setlist.md`
- `MIKE'S afternoon delight setlist.md`
- `mike's club axis set list.md`
- `mike's short set list.md`
- `mike's wayout set list.md`
- `Molly Hi-Pointe.md`
- `molly's set list.md`
- `molly's short set list.md`
- `Mike Hi-Pointe.md`
- `Hi-Pointe.md`
- `open mike.md`
- `set list.md`

### 05-comic-book/
Comic book scripts and related content
- `comic help letter.md` - Letter about comic book creation
- `comicstrip (revised).md` - Revised comic strip script
- `comicstrip.md` - Original comic strip script

### 99-other/
Miscellaneous notes and documents
- `no kidz notes.md` - Various band notes
- `ALL notes.md` - Compilation of notes
- `demo tape.md` - Demo tape information (placeholder - original was corrupted)

## Source Files

Original Word documents (.doc files) remain in their original locations:
- `Documents/` - Original band documentation
- `Comic Book/` - Original comic book files and assets

## Usage

This organized content is ready to be integrated into the Plutonium Kidz website. See the main website integration plan for how to incorporate these files into HTML pages.
